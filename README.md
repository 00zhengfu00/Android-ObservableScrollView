Android-ObservableScrollView
===

## Installation

```groovy
dependencies {
    compile 'com.github.ksoichiro:android-observablescrollview:1.0.0'
}
```

## Samples

See `observablescrollview-samples` directory.

To install:

```sh
$ ./gradlew installDebug
```

## Supported widgets

* ListView
* ScrollView
* WebView
